### create new session:
```bash
tmux new -s "mysession"
```

### Attach to existings session
```bash
tmux ls
tmux at
tmux at -t mysession
```

### Create new pane
```bash
Ctrl+f, c
```

### Detach (minimize) session
```bash
Ctrl+f, d
```

### Split screen
```bash
Ctrl+f, -
Ctrl+f, \
```

### Kill pane in session
```bash
Ctrl+d
```

### Copy-paste
Hold Shift (linux) or Fn (Mac) and copy-past like usual

### Scroll
```bash
Ctrl+f, [
#(Arrow up/down)
# q to quit
```
